Data Flow Steps:

  1. User Request: Heritage site visitor scans QR code on mobile device
  2. DDoS Protection: AWS Shield protects against malicious traffic
  3. Global CDN: CloudFront serves cached content or forwards to origin
  4. Security Filter: AWS WAF filters malicious requests and applies rate limiting
  5. API Gateway: Routes requests, handles authentication, and validates input
  6. Serverless Compute: Lambda functions process QR codes and generate content
  7. AI Processing: Bedrock generates content, Polly creates audio, Translate handles languages
  8. Content Delivery: Generated content stored in S3, metadata cached in DynamoDB, delivered via CloudFront
