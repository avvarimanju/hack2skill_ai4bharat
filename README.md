# Hack2Skill AI4Bharat Projects

This repository contains projects for the AWS AI for Bharat Hackathon.

## Projects

### [AvvarI for Bharat](./avvari-for-bharat)

AI-Powered Heritage Site Digitization Platform

**Description:** AvvarI transforms heritage site experiences using AWS AI services. Visitors scan QR codes to receive personalized, multilingual content including AI-generated descriptions, audio guides, and interactive Q&A.

**Key Technologies:**
- Amazon Bedrock (AI content generation)
- Amazon Polly (Text-to-speech)
- Amazon Translate (Multilingual support)
- AWS Lambda (Serverless compute)
- DynamoDB, S3, CloudFront

**Status:** ✅ Complete - 767 tests passing
